Deployment Report Builder

Date: 2021-06-16
Filename: Invoke-DeploymentReport.ps1

A deployment reporting tool that shows success, failure, in progress, and unknown objects using CIM.
The resulting report is saved on the users desktop as a CSV file, or if ImportExcel PowerShell module is installed, an Excel formatted worksheet.

The script walks through the following steps in creating the report

  1. CIM is used to gather all current deployments
  2. User is prompted to select a deployment and targeted collection
  3. The targeted collection is enumerated to account for all devices deployed to
  4. CIM is used to get the status of the current deployment (note: this excluded Unknown)
  5. The targeted collection is then enumerated, pulling in the result if found based off of resource ID
  6. A CSV file is saved to the desktop, or processing continues if ImportExcel is installed
  7. An Excel worksheet is created, sorted, highlighting status result, and freezing top 2 rows
  8. The report is saved to the desktop in XLSX format and launched.

NOTE

To remove the launching, remove the -SHOW switch
